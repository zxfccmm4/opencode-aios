# Bug Fix Template

## Bug

What is broken?

## Expected Behavior

What should happen?

## Actual Behavior

What happens instead?

## Reproduction

Steps, logs, screenshots, or failing tests.

## Suspected Area

Relevant files, modules, or systems.

## Fix Plan

- Identify root cause
- Implement fix
- Verify bug is fixed
- Check nearby regressions

## Verification

What proves the bug is fixed?
